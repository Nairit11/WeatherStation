This weather app uses the API provided by OpenWeatherMap for the data it displays.
This app can display the climate parameters of any city queried and also can calculate the average temperature of any four cities of a country.
weather.html is used to display weather conditions at the queried city.
mainWeather.js is the file which uses  the OpenWeatherMap API.
weather2.html is used to  calculate the average temperature of 4 cities.
mainWeather1.js,mainWeather6.js,mainWeather3.js,mainWeather5.js are the files called by weather2.html to gather data from OpenWeatherMap API 4 times and calculate their average.
